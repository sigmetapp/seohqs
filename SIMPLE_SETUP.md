# Простая настройка

## Что сделано:

1. ✅ Простая страница с таблицей данных
2. ✅ API для получения данных: `/api/data`
3. ✅ API для загрузки тестовых данных: `/api/seed`
4. ✅ Подключение к базе данных (PostgreSQL на Vercel, SQLite локально)

## Шаги для запуска:

### 1. Убедитесь, что база данных создана

Выполните SQL из `migrations/manual_migration.sql` в вашей PostgreSQL базе данных на Vercel.

### 2. Установите переменные окружения в Vercel

В настройках проекта Vercel добавьте:
- `DATABASE_URL` или `POSTGRES_URL` - строка подключения к PostgreSQL

### 3. Деплой

После деплоя:
1. Откройте сайт
2. Нажмите кнопку **"Загрузить тестовые данные"**
3. Данные должны появиться в таблице

## Локальный запуск:

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## Что делает каждая кнопка:

- **"Обновить данные"** - загружает данные из базы
- **"Загрузить тестовые данные"** - очищает базу и загружает 5 тестовых записей
