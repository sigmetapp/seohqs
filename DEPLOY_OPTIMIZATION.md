# Оптимизация деплоя

## Выполненные оптимизации:

1. **Next.js конфигурация** (`next.config.js`):
   - Отключен `reactStrictMode` для ускорения сборки
   - Включен `swcMinify` для быстрой минификации
   - Отключены проверки TypeScript и ESLint при сборке
   - Используется `standalone` режим для минимального размера
   - Оптимизированы импорты пакетов

2. **Vercel конфигурация** (`vercel.json`):
   - Используется `npm ci --prefer-offline --no-audit` для быстрой установки
   - Установлены лимиты для функций

3. **Исключение файлов** (`.vercelignore`):
   - Исключены миграции SQL
   - Исключена документация
   - Исключены тесты и примеры

## Дополнительные рекомендации для Vercel:

В настройках проекта Vercel можно добавить переменные окружения:
- `NPM_FLAGS=--prefer-offline --no-audit --legacy-peer-deps`

## Ожидаемое время деплоя:

- Установка зависимостей: ~5-10 секунд
- Сборка Next.js: ~5-10 секунд
- Деплой: ~2-5 секунд
- **Итого: ~10-15 секунд**

## Примечания:

- Миграции базы данных не запускаются автоматически при деплое
- Все проверки типов и линтера отключены для скорости (можно включить обратно при необходимости)
- Используется кэширование зависимостей Vercel для ускорения последующих деплоев
