# Быстрый старт

## Запуск приложения

```bash
# Установка зависимостей (уже выполнено)
npm install

# Запуск dev-сервера
npm run dev
```

Откройте браузер по адресу: **http://localhost:3000**

## Добавление своих CSV файлов

Поместите ваши CSV файлы в папку `/public/data/` с именами:
- `admitad.csv`
- `cj.csv`
- `advertise.csv`
- `clickbank.csv`

### Формат CSV

Файлы должны содержать следующие колонки (названия могут быть на русском или английском):

| Колонка | Варианты названий |
|---------|-------------------|
| Название | `name`, `название`, `program`, `program name` |
| Тематика | `topic`, `тематика`, `category`, `category name` |
| Страна | `country`, `страна`, `geo`, `geography` |
| Модель | `model`, `модель`, `commission type`, `commission_type` |
| CR | `cr`, `conversion rate`, `conversion_rate` |
| eCPC | `ecpc`, `effective cpc`, `effective_cpc` |
| EPC | `epc`, `earnings per click`, `earnings_per_click` |

## Использование

1. **Фильтрация**: Используйте выпадающие списки и поле поиска для фильтрации данных
2. **Сортировка**: Кликайте на заголовки столбцов для сортировки
3. **Пагинация**: Используйте кнопки "Назад" и "Вперёд" для навигации
4. **Экспорт**: Нажмите "Export CSV" для экспорта отфильтрованных данных
