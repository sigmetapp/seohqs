# Affiliate Catalog

Одностраничное веб-приложение для работы с каталогом партнёрских программ (affiliate offers).

## Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **TanStack React Table** - таблица с сортировкой
- **PapaParse** - парсинг CSV файлов

## Возможности

✅ Фильтрация по:
- Тематике
- Стране
- Модели вознаграждения (CPS, CPA, RevShare)
- Поиск по названию программы

✅ Сортировка по:
- CR (Conversion Rate)
- eCPC (Effective Cost Per Click)
- EPC (Earnings Per Click)
- Любому другому столбцу

✅ Постраничное отображение (20 записей на страницу)

✅ Автоматическое объединение данных из всех CSV-файлов

✅ Экспорт отфильтрованных данных в CSV

## Структура проекта

```
/app
├─ /components
│ ├─ Table.tsx          # Компонент таблицы с сортировкой
│ ├─ Filters.tsx        # Компонент фильтров
│ └─ Loader.tsx         # Компонент загрузки
├─ /utils

├─ page.tsx             # Главная страница

/public
└─ /data

```

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev-сервер:
```bash
npm run dev
```

3. Откройте браузер по адресу:
```
http://localhost:3000
```

## Формат CSV файлов

CSV файлы должны содержать следующие колонки:
- `name` - название программы
- `topic` - тематика
- `country` - страна
- `model` - модель вознаграждения
- `cr` - Conversion Rate
- `ecpc` - Effective Cost Per Click
- `epc` - Earnings Per Click

Утилита автоматически нормализует названия колонок (case-insensitive) и обрабатывает различные варианты названий полей.

## Использование

1. При загрузке приложение автоматически объединяет все CSV файлы из папки `/public/data`
2. Используйте фильтры для поиска нужных программ
3. Кликайте на заголовки столбцов для сортировки
4. Используйте пагинацию для навигации по страницам
5. Нажмите "Export CSV" для экспорта отфильтрованных данных
